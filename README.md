# Admin Tools

**Admin Tools** рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃФрЃџрЃўрЃћрЃарЃў Lua рЃАрЃЎрЃарЃўрЃърЃбрЃў EDRP-рЃА рЃљрЃЊрЃЏрЃўрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃўрЃА, рЃљрЃЋрЃбрЃЮ-рЃерЃћрЃАрЃЋрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃў GUI рЃЏрЃћрЃюрЃўрЃБрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃћрЃЉрЃА.

---

## Рюе рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў

- ­Ъћљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃАрЃЋрЃџрЃљ `/alogin`-рЃерЃў (рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃЌ)
- РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃћрЃюрЃўрЃБ `/amenu` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЌ
- РюЁ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА `/az` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ
- ­Ъњг рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў Quick Reply GUI рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃдрЃћрЃЉ report-рЃА
- ­ЪЉЂ№ИЈ рЃћрЃарЃЌрЃў рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ `рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃўрЃА` рЃЊрЃљ `Recon-рЃўрЃА` рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў (`/a [REPORT] (рЃарЃћрЃърЃЮрЃарЃбрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў)` рЃЊрЃљ `/re`)
- ­ЪњЙ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ `admin_tools.ini` рЃцрЃљрЃўрЃџрЃерЃў
